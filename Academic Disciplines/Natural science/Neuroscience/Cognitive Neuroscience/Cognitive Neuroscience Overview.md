# Cognitive Neuroscience Overview

Cognitive neuroscience investigates the neural mechanisms underlying cognitive processes such as perception, attention, memory, language, reasoning, and consciousness.

## Topics

### Perception and Attention
- Visual and auditory processing
- Attention networks and mechanisms
- Integration of sensory information

### Memory Systems
- Encoding, consolidation, and retrieval
- Short-term and working memory
- Long-term memory and consolidation
- Memory disorders

### Language
- Language processing networks
- Speech production and comprehension
- Reading and writing
- Language development and plasticity

### Executive Functions
- Decision making and reasoning
- Cognitive control and inhibition
- Goal-directed behavior

### Consciousness and Awareness
- Neural correlates of consciousness
- Sleep and anesthesia
- Altered states of consciousness
