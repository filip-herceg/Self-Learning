# Computational Neuroscience Overview

Computational neuroscience uses mathematical and computational methods to model neural systems, understand information processing in the brain, and generate testable predictions about neural function.

## Topics

### Neural Modeling
- Hodgkin-Huxley models
- Integrate-and-fire neurons
- Biophysical neuron models
- Cable theory

### Network Modeling
- Artificial neural networks
- Spiking neural networks
- Recurrent network dynamics
- Learning algorithms

### Population Coding
- Neural population decoding
- Information theory in neuroscience
- Representation and coding schemes

### Data Analysis
- Time series analysis
- Dimensionality reduction
- Statistical inference in neuroscience

### Brain Simulation
- Large-scale brain models
- Digital brain projects
- Virtual neural systems
