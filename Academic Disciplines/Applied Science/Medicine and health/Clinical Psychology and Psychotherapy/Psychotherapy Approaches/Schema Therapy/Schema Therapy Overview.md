# Schema Therapy Overview

Schema Therapy integrates cognitive-behavioral, attachment, experiential, and psychodynamic approaches to address deeply-rooted, enduring patterns (schemas) and maladaptive coping modes.

## Theoretical Foundations

- Schema theory (early maladaptive schemas)
- Modes of functioning
- Attachment needs
- Emotional deprivation and frustration
- Cognitive-behavioral integration

## Core Concepts

### Early Maladaptive Schemas
- Definitions and characteristics
- Schema domains
- Development and origins
- Life patterns

### Eighteen Core Schema Domains

**Disconnection and Rejection**
- Abandonment
- Mistrust/Abuse
- Emotional Deprivation
- Defectiveness
- Social Isolation

**Impaired Autonomy and Performance**
- Dependence/Incompetence
- Vulnerability to Harm
- Enmeshment
- Failure

**Impaired Limits**
- Entitlement
- Insufficient Self-Control

**Other-Directedness**
- Subjugation
- Self-Sacrifice
- Approval-Seeking

**Overvigilance and Inhibition**
- Negativity/Pessimism
- Emotional Inhibition
- Unrelenting Standards
- Punitiveness

### Coping Modes
- Surrenderer mode
- Protector mode
- Detached mode
- Demanding parent mode
- Vulnerable child mode
- Healthy adult mode
